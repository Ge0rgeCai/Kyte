## 安装

如果你使用git则在终端中执行以下代码

```bash
git clone https://github.com/Ge0rgeCai/Kyte.git
```

Github CLI用户可以在终端中执行以下代码

```bash
gh repo clone Ge0rgeCai/Kyte
```
